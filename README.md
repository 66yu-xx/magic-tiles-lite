# Magic Tiles Lite v7

本版重点修正核心手感：

1. 判定线和灰色判定区整体上移半个灰框高度。
2. 点到白线以上的方块不再直接 Perfect。
3. 所有得分都按点击时间和方块目标时间的误差判断：
   - ≤ 0.07 秒：Perfect
   - ≤ 0.14 秒：Good
   - ≤ 0.22 秒：OK
   - 超过 0.22 秒：Miss
4. 太早点击无效，不加分、不扣分、不清 combo。
5. 方块漏过目标时间 0.22 秒后，自动 Miss。
6. 仍然保留暂停 / 继续 / 重开 / 退出。
